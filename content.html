2016巴黎車展：Mercedes-AMG GT C Roadster定裝照亮相

撰文│吳加男
點閱數：1,043 次
2016/09/15
facebook
plurk

在先前，U-CAR曾報導即將在巴黎車展現身的Mercedes-AMG GT C Roadster車系，所釋出的預告版影片，如今，原廠正式公布了Mercedes-AMG GT C Roadster的定裝照片，同時將會有GT Roadster與GT C Roadster兩個車型版本，分別可輸出462匹以及510匹馬力。

Mercedes原廠正式公布了GT C Roadster的定裝照片，直柵式水箱罩設計則與GT R相同。
分有GT Roadster與GT C Roadster兩款車型，動力輸出分別為462匹以及510匹馬力。

Mercedes-AMG GT C Roadster延續了品牌敞篷車系的特色，有著類似SLS、SLR的設計感，同時GT C Roadster因應其敞篷機構的需求，針對車尾部分進行修改，不過整體而言還是承襲AMG GT車系的主要元素，讓人一眼就能認出。

GT C Roadster採用與稍早發表的Mercedes-AMG GT R相同的直柵式水箱罩設計，敞篷機構方面，則是Mercedes-AMG 全新設計的軟頂敞篷結構，以鋁鎂合金搭配3層織布材質製造，據原廠表示除了重量得以減輕，在車速50公里以下時，也僅需要11秒就能完成開篷或關蓬的程序，同時車主還得以選配黑色以外的軟蓬顏色。

GT C Roadster因應其敞篷機構的需求，針對車尾部分進行修改，不過整體而言還是承襲AMG GT車系的主要元素，讓人一眼就能認出。
在車速50公里以下時，僅需11秒就能完成開篷或關蓬的程序，同時車主還得以選配黑色以外的軟蓬顏色。

Mercedes-AMG GT C Roadster將會推出GT Roadster、及GT C Roadster兩個車型版本，均搭載4.0升V8雙渦輪引擎與AMG Speedshift DCT 7速雙離合器自手排變速箱，較為入門的GT Roadster車重為1,595公斤，可輸出462匹馬力與 61.2公斤米扭力，靜止加速至時速100公里為4 秒，極速為302公里。

另一款較高定位的GT C Roadster版本車身重量為1,660公斤，可輸出510匹馬力與 66.3公斤米扭力，由靜止加速至時速100公里為3.7秒，極速為316公里。由此可知，GT Roadster與GT C Roadster的動力將比照Coupé版本的GT與GT S。

不過，GT C Roadster將擁有與GT R相同的主動式後輪轉向系統，車輛會依照駕駛狀況自動微調後輪束角最多達1.5度，來同步加強高速穩定性和過彎靈活性，這是Coupé所沒有的。

GT C Roadster將擁有與GT R相同的主動式後輪轉向系統，有助於車輛動態表現。

從發表的內裝照片來看，GT C Roadster提供了Alcantara麂皮、Nappa 皮革、碳纖維飾版、亞光銀色飾版、鋼琴漆飾版等不同的標配與選配，同時車主也可選擇車內色調的配置，延續了品牌豪華感。
